# COMMON PARAMETER NAMING SCHEMES
### BUILTIN
```
st -- Most likely a `str`
txt - Most likely a `str`
num - Most likely an `int` or `float`
dic - Most likely a `dict`
ls -- Most likely a `list`
tp -- Most likely a `tuple`
```

### CLASSES
```
chnl --- Text Channel
vc ----- Voice Channel
cat ---- Catagory Channel
gld ---- Guild
emj ---- Emoji
plr ---- Player
mbr ---- Player
msg ---- Message
usr ---- User
rol ---- Role
rct ---- Reaction
web ---- Webhook
```
